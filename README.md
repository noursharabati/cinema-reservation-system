# Cinema Reservation System

A relational database project developed to manage cinema operations, including movies, showtimes, customers, halls, and ticket reservations.

## Project Overview

The system was designed to support cinema management by organizing movie schedules, customer information, ticket purchases, and hall assignments. The project demonstrates database design principles, relational modeling, and SQL implementation.

## Technologies Used

* SQL
* Database Design
* ERD Modeling
* Relational Database Concepts

## Features

* Movie management
* Showtime scheduling
* Hall management
* Customer registration
* Ticket reservation and purchase tracking
* Reporting and ticket sales analysis

## Database Components

* Movies
* ShowTimes
* Halls
* Customers
* Tickets

## Concepts Applied

* Entity Relationship Diagram (ERD)
* Relational Database Design
* Primary Keys and Foreign Keys
* Referential Integrity Constraints
* SQL Queries
* Aggregate Functions

## Project Deliverables

* Database Design Documentation
* SQL Implementation
* Presentation Slides
